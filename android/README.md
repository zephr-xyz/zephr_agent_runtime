# Android SDK

This directory is the Gradle workspace for the Android SDK.

The workspace contains one module:

- `sdks/zephr_agent_runtime`: Android wrapper around the shared Zephr Agent C API.

The shared C++ implementation remains in `../tinylib`. The Android module links
it through JNI/CMake and uses the native dependency bridge generated by
repo-level setup commands.

## Setup

From the repository root:

```bash
uv sync
uv run prepare_platform_tools
uv run prepare_native_deps --platform android
uv run prepare_dev --platform android
```

Open this directory as the Android Studio project, or build from the command
line:

```bash
cd android
./gradlew :sdks:zephr_agent_runtime:assembleDebug
```
